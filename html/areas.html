Areas

El Scesi se compone de un total de 10 areas en las cuales puedes trabajar y especializarte

Web Basico

Web basico

El area de web basico esta dedicada al diseño y estructura de paginas web, sus herramientas principales son html el cual es un lenguaje para estructurar paginas web, css para dar estilos a estas paginas y tambien js para funciones mas complejas

Web Avanzado

Web Avanzado

En web avanzado se ven y se utilizan herramientas mas complejas para la creacion de una pagina web, se utilizaran principalmente las herramientas que usan JavaScript como por ejemplo Angular

Mobile

Mobile

En mobile podras especializarte en el desarrollo de aplicaciones para dispositivos mobiles y sus sistemas operativos respectivos, una herramienta a utilar es el android studio

Base de datos

Base de datos

En base de datos se trabaja en el manejo de datos que pertenescan a un mismo proyecto o conjunto, y que sean almacenados en un mismo sistema, una herramienta para estructurar estos sistemas es mySQL.

UI/UX

UI/UX

El area de UI-UX nos ayuda para poder decidir qué diseño es el más adecuado para lo que tenemos que observar el comportamiento del usuario: qué acciones hace, qué es lo primero que mira y qué reacciones tiene. Aquí es donde entra la UX (experiencia de usuario) y la UI (interfaz de usuario), que tienen la función de que el usuario encuentre lo que busca en el menor tiempo posible y además tenga una buena experiencia al visitar la web.

Seguridad

Seguridad

El area de seguridad esta dedicada principalmente a la prevencion y deteccion de vulnerabilidades dentro los sistemas informaticos especialmente en la red, para poder decidir qué diseño es el más adecuado.

Robotica Basica

Robotica basica

Esta area comprende principalmente a introducir el mundo de la robotica, se enseña el uso del arduino y su programacion por el mismo, el armado de circuitos no muy complejos y el estudio de los componentes basicos que generalmente se usan.

Robotica Avanzada

Robotica Avanzada

El area de robotica avanzada se dedica a realizar estructuras mas complejas de circuitos, tambien se aprende el uso adecuado de componentes mas avanzados.

IA

IA

El area de inteligencia artificial es donde se estudia lo que es una ciencica nueva la cual es cambiante y experimental. Lo que abarca es intentar imitar la inteligencia humana usando software.

Backend

Backend

El backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Se trata del conjunto de acciones que pasan en una web pero que no vemos como, por ejemplo, la comunicación con el servidor.
